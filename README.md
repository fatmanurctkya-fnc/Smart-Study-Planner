# Smart Study Planner System

## Project Description
Smart Study Planner is a web-based application designed to help students manage courses, assignments and weekly study plans efficiently.

The system includes:
- Course management
- Task and deadline tracking
- Weekly study plan generation
- Time management support

The Minimum Viable Product of our project includes the following core features:

Loing System:
Users can log in using a username and password
Task management:
Users can create study plans and add specific tasks to each plan.
Progress Tracking:
The system calculates progress based on completed tasks and displays it as a percentage

We implemented CI/CD using GitHub Actions.
Every time we push code,the system automattically runs a build check.
This ensures that project does not contain errors.
It helps us keep the code in a stable and working condition at all times.

Since our project is based on Python,the pipeline can run Python scripts to verify functionality
